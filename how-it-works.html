How it works

Two weeks to launch. Clear consent. Easy redemption. Real results. Set it once and let the automations work for you.

Automatic messages that run themselves

Program One: Continuous Engagement

We schedule a welcome sequence that starts 6 hours after join, then 12 monthly messages with a 10 percent reward. Set once. No work needed.

Program Two: “We Miss You”

When a member has not checked in, the system sends messages at 20, 40, 60, 80, and 100 days with rising rewards: 5, 10, 15, 20, and 35 percent. Resets after the next check in.

Program Three: Electronic Punch Card

Reward every 6th visit. After each 5th visit, members receive a 10 percent reward to use on the 6th visit.

Program Four: Two Annual Rewards

Send a birthday perk and a join date anniversary perk each year. Simple to set up. Keeps members engaged.

Program Five: Check In Tracker

Track every visit for powerful insights. Use it to understand frequency and to trigger rewards.

Program Six: Review Request

Ask for a review after the 2nd check in. Link goes straight to your review page to make it easy.

Program Seven: Random Texts

Send custom messages at any time for sales, promos, events, and news.

Getting started

1. Grow your list

3. Track and learn
